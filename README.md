# ZMK Layer Overlay

**ZMK Layer Overlay** は、自作キーボード（特に Charybdis や分割キーボード）のレイヤー操作を視覚的にサポートするツールです。
特定のキー（レイヤー切り替えキー）を押すと、PC画面上に現在のレイヤーのキーマップ画像を一定時間オーバーレイ表示します。

## 🌟 主な特徴

* **リアルタイム表示:** レイヤー切り替えキーを押した瞬間に画像を高速表示。
* **システムトレイ常駐:** タスクバーの通知領域に常駐し、バックグラウンドで動作します。
* **表示時間のカスタマイズ:** 画像を表示する秒数を、トレイメニューから 0.1 秒単位で自由に変更可能。
* **動作の切り替え:** ツールを終了させずに、トレイメニューからいつでも「有効/無効」を切り替え可能。
* **アクティブモニター追従:** 現在マウスカーソルがある、またはアクティブなウィンドウがあるモニターの中央に表示。
* **自動リサイズ:** モニターの解像度に合わせて、画像のアスペクト比を維持したまま最適化（画面の80%サイズ）。
* **背景透過対応:** 透過PNGまたは特定の背景色を指定することで、デスクトップの邪魔をせずにキーマップを確認可能。

---

## 🚀 使い方

1. **起動:** `layer_viewer.exe` を実行すると、システムトレイ（タスクバー右下）にアイコンが表示されます。
2. **画像の表示:** 設定したキー（F13など）を押すと、中央にレイヤー画像が表示され、設定された秒数後に自動で消えます。
3. **設定変更:** トレイアイコンを**右クリック**すると以下の操作が可能です。
    - **有効/無効:** チェックを外すと、キーを押しても画像が表示されなくなります（一時停止）。
    - **表示秒数の設定:** 入力ダイアログに数値を入力して表示時間を変更できます（例: `0.8` や `4.0`）。
    - **終了:** ツールを完全に終了します。

## 🛠️ ZMK側の設定

ZMKでレイヤーを切り替える際、同時に PC が検知できる信号（F13〜F24など）を送信するマクロを設定します。

`keymap` ファイルに以下のようなマクロを定義してください。

```c
/ {
    macros {
        // レイヤー1へ切り替えつつ、F13を送信するマクロ
        mo_l1: mo_l1 {
            compatible = "zmk,behavior-macro";
            #binding-cells = <0>;
            wait-ms = <0>;
            tap-ms = <0>;
            bindings
                = <&macro_press &mo 1 &kp F13>
                , <&macro_pause_for_release>
                , <&macro_release &mo 1 &kp F13>;
        };
    };

    keymap {
        base_layer {
            bindings = <
                // 既存の &mo 1 を &mo_l1 に置き換え
                ... &mo_l1 ...
            >;
        };
    };
};
```

## 🚀 PC側のセットアップ

1. フォルダ構成
本ツールを動作させるために、以下の構成でファイルを配置してください。EXE本体と layers フォルダを同じ場所に置くことが必須です。

```
.
├── layer_viewer.exe  (または .py スクリプト)
└── layers/           (画像格納フォルダ)
      ├── f13.png     (レイヤー1用)
      ├── f14.png     (レイヤー2用)
      └── ...
```

2. 画像の準備
    - layers フォルダの中に、ZMKで設定したキー名（f13.png など）で画像を保存します。
      
    - 背景透過: 背景を透明にしたい場合は、透過PNGを使用するか、画像背景を #abcdef (スクリプト内指定色) で塗りつぶしてください。


3. Windows起動時の自動実行設定（推奨）
PCを起動した際、自動的に本ツールが立ち上がるように設定します。
    1. 任意のフォルダ（例: C:\Users\YourName\Documents\ZMK_Overlay）に本ツールのフォルダを置きます。
   
    1. layer_viewer.exe を右クリックし、**「ショートカットの作成」**を選択します。
   
    1. Win + R キーを押し、入力欄に shell:startup と入力して実行します。
   
    1. 開いたスタートアップフォルダの中に、作成したショートカットを移動します。
   
      Note: EXE本体ではなく「ショートカット」を登録することで、画像ファイル（layers フォルダ）の読み込みパスを正しく維持しつつ、安全に管理できます。

## 📦 ビルド方法 (開発者向け)
Python環境で以下のライブラリをインストールし、PyInstallerでEXE化できます。

## 必要ライブラリのインストール
pip install screeninfo pygetwindow Pillow pynput pystray

## EXE化 (コンソール非表示・1ファイルに集約)
pyinstaller --noconsole --onefile layer_viewer.py

## 📋 動作要件
OS: Windows 10 / 11

Keyboard: ZMKファームウェアを搭載したキーボード（Charybdis等）
